# EveryBonusMarkerNFSMW

A plug-and-play ASI plugin for Need for Speed: Most Wanted (2005) that allows you to claim all Bonus Markers after defeating a Blacklist Rival.

This project was created for players who want the ability to claim all markers but prefer to keep their base game as lean as possible. It serves as a lightweight alternative to [NFSMW Extra Options](https://github.com/ExOptsTeam/NFSMWExOpts/tree/master) for those who only want the bonus marker feature without the other modifications bundled with it.

The plugin replicates the exact behavior of [this Cheat Engine script by nlgxzef](https://gist.github.com/nlgxzef/bdef0aa9c48f9f6254c1fae44387caf6), but operates as a seamless drop-in modification, completely removing the need to use Cheat Engine.

## Requirements

* An ASI Loader, such as [Ultimate ASI Loader](https://github.com/ThirteenAG/Ultimate-ASI-Loader).


## Installation

1. Go to the [Releases](https://github.com/RafaelRS04/EveryBonusMarkerNFSMW/releases) page and download the latest compiled `.asi` file.
2. Place the downloaded file into the `scripts` folder located in your game directory (if you are using Ultimate ASI Loader, create the folder if it does not exist).
3. Run the game[^1].

## Building from Source

If you prefer to compile the plugin manually instead of downloading the pre-compiled binary:

1. Clone this repository.
2. Open the project solution file using Visual Studio.
3. Build the project to generate the `.asi` file.

[^1]: Tested on Need for Speed: Most Wanted (2005) Black Edition v1.3.
